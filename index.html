Projects

A web app for tracking rock climbing.

An application for rock climbers to track and view stats about attempts they make on rock climbing routes. Users are also able to create profiles, view locations, add routes, as well as tag and comment on routes.

Made with Ruby, Rails, Postgres, Haml, JavaScript, hooks into the Facebook API with the omniauth-facebook gem, geolocation services are done with the geocoder gem.

check it out on github

Homework Tracker App.

An application for submitting homework assignments for review. Users can sign in as either a teacher or student. As a teacher, you can create assignments, view students submissions and comment on them, as a student you can join a course view all assignments for that course and make submissions.

Made with Ruby, Rails, Postgres, Haml, CSS, JavaScript.
