A web app for tracking rock climbing.

an app for rock climbers to track and view stats about attempts they make on rock climbing routes. users are also able to create profiles, view locations, add routes, and tag and comment on routes.

Made with Ruby, Rails, Postgres, Haml, JavaScript, Hooks into the Facebook API with the omniauth-facebook gem, geolocation services are done with the geocoder gem.
